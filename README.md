# 📌 Project Overview

This project showcases how to build reusable UI components in React. It is designed with responsive layouts, modern CSS styling, hover animations, and component-based architecture.

## Features

- Dashboard Component
- Employee Card Component
- Development Team Component
- Responsive Design
- Modern CSS Styling
- Reusable React Components

## Technologies Used

- React.js
- JSX
- JavaScript (ES6)
- CSS3
- HTML5

### Dashboard

- Team Information
